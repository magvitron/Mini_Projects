The RAR file includes 2 Simulink models, 3 *.m file and one icon folder
- Model_14Bus_Microgrid_R2018.slx
- Model_14Bus_Microgrid_R2017b.slx
- Set_parameters_R2018.m
- Set_parameters_R2017b.m
- Save_Data.m
- Graphics.m
- Demand.m
- Icons folder

Steps in order to RUN the Model:  
1- Copy and unzip the WinRAR file in a new folder.
2- Open and RUN the Simulink version according to your Matlab version.
3- Finally, the Simulink File automatically executes the m files.

Product requirements:
- Simulink
- Stateflow
- Simscape
- Simscape Power Systems
